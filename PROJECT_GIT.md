# zcprotobuf Git Project Setup

`zcprotobuf` currently lives inside a larger workspace.  
Use the auto publish script:

```bash
cd zcprotobuf
./publish_github.sh
```

Optional:

```bash
./publish_github.sh your-repo-name
./publish_github.sh your-account/your-repo-name
VISIBILITY=private ./publish_github.sh
```
